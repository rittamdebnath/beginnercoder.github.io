Downloading & Installing packages

To install packages you can goto Packages -> Settings View - > Install Packages/Theme or you can press Ctrl + , then Click on install and then search the following packages in the search bar on the right and press install.

  1. Atom Editor
  2. Emmet
  3. atom beautify
  4. color picker
  5. atom-bootstrap3

Good luck your ready to go !!!
